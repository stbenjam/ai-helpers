# Test Coverage Plugin

Analyze e2e/integration test code structure without running tests to identify coverage gaps.

## Commands

### `/test-coverage:analyze`

Analyze e2e/integration test code structure without running tests to identify files and functions without test coverage.

### `/test-coverage:gaps`

Identify missing test coverage in OpenShift/Kubernetes test files with intelligent gap analysis for any component.

See the [commands/](commands/) directory for full documentation of each command.

## Installation

```bash
/plugin install test-coverage@ai-helpers
```
